# 随机过程教程

## 课程目标
本课程旨在为机器人专业学生提供概率论和随机过程的理论基础，重点介绍随机过程的核心概念及其在机器人状态估计、滤波和决策中的应用。通过本课程学习，学生将能够：
1. 掌握概率论的基本理论和方法
2. 理解随机过程的核心概念和性质
3. 熟悉马尔可夫过程、平稳过程等重要随机过程
4. 能够将随机过程方法应用于机器人定位、导航和控制问题

## 课程内容

### 第一部分：概率论基础（16学时）

#### 第1章：概率空间和随机变量（4学时）
1.1 概率空间的定义
- 样本空间、事件域和概率测度
- 概率的公理化定义和基本性质
- 条件概率和全概率公式

1.2 随机变量的概念
- 随机变量的定义和分类
- 离散型和连续型随机变量
- 随机变量的分布函数和概率密度函数

1.3 常见概率分布
- 离散分布：二项分布、泊松分布、几何分布
- 连续分布：正态分布、均匀分布、指数分布
- 多维随机变量的联合分布和边缘分布

#### 第2章：随机变量的数字特征（4学时）
2.1 数学期望和方差
- 数学期望的定义和性质
- 方差和标准差的概念
- 协方差和相关系数

2.2 矩和特征函数
- 原点矩和中心矩
- 特征函数的定义和性质
- 特征函数在概率计算中的应用

2.3 大数定律和中心极限定理
- 大数定律的基本形式
- 中心极限定理的内容和应用
- 大数定律在机器人中的应用

#### 第3章：条件概率和贝叶斯理论（4学时）
3.1 条件概率和条件期望
- 条件概率的定义和性质
- 条件期望的计算方法
- 条件期望的性质和应用

3.2 贝叶斯公式
- 贝叶斯公式的推导和解释
- 贝叶斯推断的基本思想
- 贝叶斯方法在机器人中的应用

3.3 随机变量的独立性
- 随机变量独立性的定义
- 独立随机变量的性质
- 独立性的判定方法

#### 第4章：随机向量和随机过程基础（4学时）
4.1 随机向量的概念
- 随机向量的联合分布
- 随机向量的数字特征
- 随机向量的变换

4.2 随机过程的定义
- 随机过程的基本概念
- 随机过程的分类：离散时间和连续时间
- 随机过程的数字特征

4.3 随机过程的基本类型
- 独立增量过程
- 马尔可夫过程
- 平稳过程

### 第二部分：随机过程理论（20学时）

#### 第5章：马尔可夫过程（6学时）
5.1 马尔可夫链的定义和性质
- 马尔可夫性的定义
- 转移概率矩阵和状态转移图
- 马尔可夫链的分类：常返态和暂态

5.2 马尔可夫链的极限分布
- 平稳分布的定义和性质
- 极限分布的计算方法
- 遍历性定理

5.3 连续时间马尔可夫过程
- 连续时间马尔可夫链的定义
- 转移速率矩阵
- 生灭过程和排队模型

5.4 马尔可夫过程在机器人中的应用
- 机器人状态转移模型
- 马尔可夫决策过程
- 机器人路径规划中的马尔可夫方法

#### 第6章：平稳过程和遍历性（6学时）
6.1 平稳过程的定义
- 严平稳和宽平稳过程
- 平稳过程的自相关函数
- 平稳过程的功率谱密度

6.2 遍历性理论
- 遍历性的定义和判定
- 遍历性定理的内容
- 遍历性在机器人中的应用

6.3 平稳过程的线性变换
- 线性系统对平稳过程的响应
- 滤波和预测的基本概念
- 维纳滤波的基本思想

#### 第7章：泊松过程和布朗运动（8学时）
7.1 泊松过程的定义和性质
- 泊松过程的基本定义
- 泊松过程的数字特征
- 泊松过程的叠加和分解

7.2 非齐次泊松过程
- 非齐次泊松过程的定义
- 强度函数的概念
- 非齐次泊松过程的应用

7.3 布朗运动的定义和性质
- 布朗运动的基本定义
- 布朗运动的数字特征
- 布朗运动的性质和应用

7.4 布朗运动的推广
- 多维布朗运动
- 布朗桥和其他扩散过程
- 布朗运动在机器人中的应用

### 第三部分：随机过程在机器人中的应用（12学时）

#### 第8章：概率机器人基础（4学时）
8.1 概率机器人的基本概念
- 机器人的不确定性来源
- 概率机器人的基本框架
- 状态估计和滤波的必要性

8.2 机器人的概率模型
- 运动模型：状态转移概率
- 观测模型：传感器测量概率
- 环境模型的概率表示

8.3 贝叶斯滤波的基本框架
- 贝叶斯滤波的递推公式
- 预测和更新步骤
- 贝叶斯滤波的收敛性分析

#### 第9章：状态估计算法（4学时）
9.1 卡尔曼滤波
- 线性系统的卡尔曼滤波
- 卡尔曼滤波的推导和解释
- 卡尔曼滤波的扩展：EKF和UKF

9.2 粒子滤波
- 粒子滤波的基本思想
- 重要性采样和重采样
- 粒子滤波的改进算法

9.3 其他滤波方法
- 扩展卡尔曼滤波
- 无味卡尔曼滤波
- 信息滤波

#### 第10章：随机控制和决策（4学时）
10.1 随机控制的基本概念
- 随机控制系统的定义
- 随机最优控制的基本框架
- 动态规划在随机控制中的应用

10.2 马尔可夫决策过程
- MDP的定义和要素
- 值迭代和策略迭代算法
- 部分可观测马尔可夫决策过程（POMDP）

10.3 机器人决策和规划
- 基于随机过程的路径规划
- 机器人导航中的决策方法
- 多机器人协调的随机决策

## 实验环节
1. **实验一**：概率论基础实验（6学时）
   - 实现常见概率分布的模拟
   - 验证大数定律和中心极限定理
   - 实现贝叶斯推断算法

2. **实验二**：马尔可夫过程实验（6学时）
   - 实现马尔可夫链的模拟
   - 计算马尔可夫链的平稳分布
   - 实现马尔可夫决策过程的求解

3. **实验三**：机器人状态估计实验（6学时）
   - 实现卡尔曼滤波算法
   - 实现粒子滤波算法
   - 比较不同滤波算法在机器人定位中的性能
