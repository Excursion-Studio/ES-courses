# 第8章 曲率理论

## 8.1 曲率的概念

### 8.1.1 曲率的直观理解

曲率是描述空间弯曲程度的几何量。在欧几里得空间中，直线是"平直"的（曲率为零），而圆是"弯曲"的（曲率非零）。在一般的黎曼流形上，曲率的概念更加复杂，涉及到多个方向上的弯曲。

**例 8.1（曲线的曲率）** 平面曲线 $\gamma(t)$ 的曲率为：
$$\kappa = \frac{|\dot{x}\ddot{y} - \dot{y}\ddot{x}|}{(\dot{x}^2 + \dot{y}^2)^{3/2}}$$

对于半径为 $R$ 的圆，曲率为 $\kappa = 1/R$。

**例 8.2（曲面的高斯曲率）** 曲面的**高斯曲率** $K$ 是两个主曲率的乘积。球面的高斯曲率为正常数，平面为0，双曲曲面为负。

### 8.1.2 黎曼曲率张量

**定义 8.1（黎曼曲率张量）** 设 $(M, g)$ 是黎曼流形，$\nabla$ 是列维-奇维塔联络。**黎曼曲率张量** $R$ 是 $(1, 3)$ 型张量，定义为：
$$R(X, Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X, Y]} Z$$

在坐标下，记为 $R^l_{ijk}$：
$$R\left(\frac{\partial}{\partial x^i}, \frac{\partial}{\partial x^j}\right)\frac{\partial}{\partial x^k} = R^l_{ijk} \frac{\partial}{\partial x^l}$$

**定理 8.1（黎曼曲率的坐标表达式）**
$$R^l_{ijk} = \frac{\partial \Gamma^l_{jk}}{\partial x^i} - \frac{\partial \Gamma^l_{ik}}{\partial x^j} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}$$

**定义 8.2（协变黎曼曲率张量）** 通过降指标得到 $(0, 4)$ 型张量：
$$R_{ijkl} = g_{im} R^m_{jkl}$$

### 8.1.3 曲率张量的对称性

**定理 8.2** 黎曼曲率张量满足以下对称性：

1. **反对称性**：
   $$R_{ijkl} = -R_{jikl} = -R_{ijlk}$$

2. **第一Bianchi恒等式**：
   $$R_{ijkl} + R_{iklj} + R_{iljk} = 0$$

3. **对称性**：
   $$R_{ijkl} = R_{klij}$$

4. **第二Bianchi恒等式**：
   $$\nabla_m R_{ijkl} + \nabla_k R_{ijlm} + \nabla_l R_{ijmk} = 0$$

### 8.1.4 截面曲率

**定义 8.3（截面曲率）** 设 $\sigma \subset T_p M$ 是一个二维平面（由两个线性无关向量 $X, Y$ 张成），**截面曲率**定义为：
$$K(\sigma) = \frac{R(X, Y, X, Y)}{g(X, X)g(Y, Y) - g(X, Y)^2} = \frac{R_{ijkl}X^i Y^j X^k Y^l}{\|X \wedge Y\|^2}$$

截面曲率完全确定了黎曼曲率张量。

**例 8.3（常曲率空间）**
- 欧几里得空间 $\mathbb{R}^n$：$K = 0$
- 半径为 $R$ 的球面 $S^n$：$K = 1/R^2$
- 双曲空间 $\mathbb{H}^n$：$K = -1/R^2$

## 8.2 曲率的计算

### 8.2.1 里奇曲率和标量曲率

**定义 8.4（里奇曲率张量）** 对黎曼曲率张量进行缩并，得到**里奇曲率张量**：
$$R_{ij} = R^k_{ikj} = g^{kl} R_{kilj}$$

**定义 8.5（标量曲率）** 进一步缩并得到**标量曲率**：
$$R = g^{ij} R_{ij} = g^{ij} g^{kl} R_{kijl}$$

**定义 8.6（爱因斯坦张量）** **爱因斯坦张量**定义为：
$$G_{ij} = R_{ij} - \frac{1}{2} g_{ij} R$$

爱因斯坦场方程为 $G_{ij} = 8\pi T_{ij}$，其中 $T_{ij}$ 是应力-能量张量。

### 8.2.2 具体流形的曲率计算

**例 8.4（球面 $S^2$）** 在球坐标下，高斯曲率为：
$$K = \frac{1}{R^2}$$

**例 8.5（柱面）** 柱面 $x^2 + y^2 = R^2$ 的高斯曲率为 $K = 0$（局部等距于平面）。

**例 8.6（环面）** 环面 $T^2$ 的高斯曲率在不同点处可正、可负、可为零。

### 8.2.3 曲率张量的计算示例

**例 8.7（二维流形）** 在二维黎曼流形上，曲率完全由高斯曲率确定：
$$R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})$$

**例 8.8（乘积流形）** 设 $M = M_1 \times M_2$，则曲率具有分块结构。

## 8.3 曲率与拓扑的关系

### 8.3.1 高斯-博内定理

**定理 8.3（高斯-博内定理）** 设 $M$ 是紧致定向二维黎曼流形，则：
$$\int_M K \, dA = 2\pi \chi(M)$$

其中 $\chi(M) = 2 - 2g$ 是欧拉示性数，$g$ 是亏格（洞的个数）。

**推论**：
- 球面 $S^2$：$\chi = 2$，$\int K \, dA = 4\pi$
- 环面 $T^2$：$\chi = 0$，$\int K \, dA = 0$
- 双环面：$\chi = -2$，$\int K \, dA = -4\pi$

### 8.3.2 里奇流

**定义 8.7（里奇流）** **里奇流**是度量张量的演化方程：
$$\frac{\partial g_{ij}}{\partial t} = -2R_{ij}$$

里奇流是微分几何中的重要工具，佩雷尔曼利用里奇流证明了庞加莱猜想。

**定理 8.4（里奇流的性质）** 在里奇流下，曲率会"扩散"，使得流形趋于均匀曲率。

### 8.3.3 曲率在机器人中的应用

**例 8.9（构型空间的曲率）** 机器人构型空间的曲率影响运动规划的几何性质。正曲率空间中的测地线可能收敛，负曲率空间中的测地线可能发散。

**例 8.10（控制中的曲率）** 在几何控制理论中，曲率影响系统的可控性和最优控制的几何结构。

## 8.4 本章小结

本章介绍了曲率理论的基本概念：

1. **黎曼曲率张量**：描述流形在多个方向上的弯曲
2. **截面曲率**：二维平面上的曲率，完全确定黎曼曲率
3. **里奇曲率和标量曲率**：曲率张量的缩并形式
4. **高斯-博内定理**：曲率与拓扑的不变量关系
5. **里奇流**：曲率的演化方程

曲率是黎曼几何的核心概念，在物理学和工程中有广泛应用。
