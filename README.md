# Excursion Studio Courses

> 远行工作室（Excursion Studio）出品的教程系列

## 📚 工作室教程系列简介

本系列教程专为机器人专业学生设计，目前涉及的领域为概率机器人学，分别包含**凸优化**、**微分几何**、**随机过程**等课程。未来将继续添加其他相关领域的教程，如强化学习、机器人控制等。

## 🎯 教程列表

### 1. [凸优化](./convex-optimization/)

从凸集凸函数的基本概念出发，深入讲解凸优化理论、多目标优化和流形优化，重点介绍其在机器人路径规划、控制和决策中的应用。

**核心内容：**
- 凸集与凸函数
- 凸优化问题与对偶理论
- 多目标优化与帕累托最优性
- 流形优化算法
- 机器人应用实例

### 2. [微分几何](./differential-geometry/)

从拓扑基础到微分流形，系统学习微分几何理论，深入理解李群李代数及其在机器人运动学和动力学建模中的应用。

**核心内容：**
- 拓扑空间与拓扑流形
- 微分流形与微积分
- 李群与李代数
- 黎曼几何与曲率理论
- 机器人运动学建模

### 3. [随机过程](./stochastic-process/)

从概率论基础到随机过程理论，掌握马尔可夫过程、滤波算法，学习如何在不确定性环境下进行机器人状态估计和决策。

**核心内容：**
- 概率论基础
- 马尔可夫过程与平稳过程
- 泊松过程与布朗运动
- 卡尔曼滤波与粒子滤波
- 马尔可夫决策过程
